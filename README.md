# 🎭 Face Emotion Recognition Web App

## 📌 Overview

This project is a **full-stack AI-powered web application** that detects human emotions from facial expressions in real-time.

---

## 🚀 Features

* 🎥 Real-time face emotion detection
* 🧠 AI-based emotion classification (Happy, Sad, Angry, etc.)
* 🌐 Interactive and responsive web interface
* 🔗 REST API built with Flask

---

## 🛠️ Tech Stack

**Frontend**

* HTML
* CSS
* JavaScript

**Backend**

* Python (Flask)

**AI / ML**

* OpenCV
* Deep Learning / Fallback Model

---

## 📂 Project Structure

```
Face-Emotion-Recognition/
│── app.py                # Flask backend
│── requirements.txt     # Dependencies
│── templates/           # HTML files
│── static/              # CSS, JS, assets
│── models/              # ML models
```

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/balaraja18/Face-Emotion-Recognition.git
cd Face-Emotion-Recognition
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the application

```bash
python app.py
```
